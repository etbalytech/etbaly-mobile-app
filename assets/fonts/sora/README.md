# Sora Font Files

Download Sora font files from https://fonts.google.com/specimen/Sora

Required files:
- Sora-Regular.ttf
- Sora-Medium.ttf (weight 500)
- Sora-SemiBold.ttf (weight 600)
- Sora-Bold.ttf (weight 700)
- Sora-ExtraBold.ttf (weight 800)
- Sora-Black.ttf (weight 900)

Download these files and place them in this directory.
